看到香港有银行投资区块链，所以了解一下相关内容。

## 公链对比

| 区块链          | Layer                  | 共识机制              | TPS（实际 / 峰值）     | 平均 Gas 费              | 主流优势                      | 典型生态 / 代表应用                  | 适合人群 / 场景                  |
|-----------------|------------------------|-----------------------|------------------------|--------------------------|-------------------------------|---------------------------------------|----------------------------------|
| Ethereum       | L1                     | PoS                   | 15 – 40 / L2 数百 – 数千 | 中高（L1 $0.5 – 10，L2 $0.005 – 0.5） | 最成熟 DeFi + RWA + 机构采用   | Uniswap, Aave, OpenSea, Base / Arbitrum | 安全性 / 机构 / 长期持有        |
| Solana         | L1                     | PoH + PoS             | 数百 – 3000+           | 极低（$0.00005 – 0.01）   | 高性能 + Memecoin + 游戏      | Jupiter, Pump.fun, DePIN, AI Agent    | 速度党 / 频繁交易 / Meme 玩家    |
| BNB Chain      | L1                     | PoSA                  | 数百 – 2000+           | 极低（$0.005 – 0.1）      | 低成本 + 大量小币 / Memecoin   | PancakeSwap, Venus, meme 项目         | 新手 / 低成本玩小币              |
| Base           | L2 (Optimistic Rollup) | Ethereum Rollup       | 数百 – 数千            | 极低（$0.005 – 0.1）      | Coinbase 导入 + 社交 / 创作者  | Aerodrome, Friend.tech 类, Onchain Summer | 美国 / 新手 / 社交型应用         |
| Arbitrum       | L2 (Optimistic Rollup) | Ethereum Rollup       | 数百 – 数千            | 极低（$0.01 – 0.2）       | DeFi TVL 王 + 成熟度高         | GMX, Uniswap V3, Treasure DAO         | DeFi 重度用户 / TVL 优先         |
| Tron           | L1                     | DPoS                  | 数百 – 2000+           | 极低（几乎免费）          | USDT / USDC 稳定币转账霸主     | SunPump meme, 大量 USDT 转账          | 稳定币大额 / 频繁转账用户        |

### 推荐
- 优先安全性与生态深度 → 选择 Ethereum 及其 L2（如 Base、Arbitrum）  。
- 追求极致速度和最低费用 → Solana 或 Tron 更合适  。
- 新手或喜欢玩小币 / Memecoin → BNB Chain 和 Base 体验最佳  。

## 钱包对比

| 钱包名称              | 类型                        | 支持链数量             | 主流优势 / 特色                        | 安全性等级   | 适合人群                          | 缺点 / 注意点                  |
|-----------------------|-----------------------------|------------------------|----------------------------------------|--------------|-----------------------------------|--------------------------------|
| Trust Wallet         | 移动热钱包                 | 非常多                | Binance 出品、多链支持、简洁易用      | 中等 - 高   | 新手、移动端、多链小额资产持有者 | 中心化团队开发                |
| MetaMask             | 热钱包（浏览器 + App）     | 非常多（EVM 为主）      | DApp 交互最方便、插件生态最全         | 中等        | DeFi、NFT、EVM 链重度玩家        | 热钱包、易钓鱼、需警惕授权   |
| Phantom              | 热钱包（Solana 为主）      | Solana + 部分 EVM     | Solana 生态最佳体验、NFT / Memecoin 强 | 中等 - 高   | Solana Memecoin、NFT、DeFi 用户  | Solana 外支持较弱             |
| OKX Wallet           | 热钱包（多端）             | 非常多                  | 聚合交易、限价单、链上赚币功能丰富   | 中等 - 高   | 交易频繁、一站式 DeFi 用户       | 国内背景、注意地区合规        |
| TronLink             | 热钱包（浏览器 + App + 插件） | Tron 为主 + 多链扩展 | Tron 官方推荐、USDT / TRC-20 支持最佳 | 中等 - 高   | Tron 生态、稳定币转账、SunPump meme 玩家 | Tron 外链支持有限、需防钓鱼 |
| Ledger               | 冷钱包（硬件）             | 非常多                | 最高安全、私钥永不联网                | 极高        | 大额长期持有、机构 / 高净值用户 | 操作稍复杂、价格较高          |

### 推荐
- 新手 / 移动端首选 → **Trust Wallet**（用户最多、简单上手）。
- DeFi / NFT / EVM 重度玩家 → **MetaMask**（生态最成熟）。
- Solana Memecoin / 游戏玩家 → **Phantom**（Solana 体验最佳）。
- 频繁交易 / 一站式 DeFi → **OKX Wallet**（功能丰富）。
- 稳定币大额转账 / Tron 生态 → **TronLink**（专精高效）。
- 大额资产安全存储 → **Ledger**（必须搭配冷钱包）。
- 通用建议：热钱包适合日常交互，冷钱包用于大额长期持有；始终警惕钓鱼、授权风险；定期备份助记词。

## 交易所对比

| 交易所   | 主流优势 / 特色                          | 现货手续费（Maker / Taker）                  | 合约手续费（Maker / Taker） | 适合人群                        | 缺点 / 风险点                  | 人民币交易                          |
|----------|------------------------------------------|-------------------------------------------------------|-------------------------------------------------------|---------------------------------|--------------------------------|-----------------------------------------|
| Binance  | 币种最多、流动性最深、生态最全          | 0.10% / 0.10%（BNB 支付可降 25%+）                   | 0.02% / 0.05%（BNB / VIP 可进一步降低）              | 全能型、重交易用户             | 监管压力大、大陆访问受限      | 支持（P2P / C2C / 快捷买币 |
| OKX      | Web3 钱包强、产品创新多、合约体验优秀  | 0.08% / 0.10%（OKB / VIP 可进一步降低）              | 0.02% / 0.05%（OKB / VIP 可降至负 Maker）            | 合约玩家、DeFi + CeFi 结合    | 国内背景、需注意合规风险      | 支持（P2P / C2C） |
| Gate.io  | 上币最快、小币种类最多                  | 0.20% / 0.20%（VIP / GT 可降至 0% / 0.02%）          | 0.015%–0.02% / 0.05%（GT / VIP 可大幅降低）          | 打新 / 小币玩家                | 安全性争议较多                | 支持（P2P / C2C） |
| MEXC     | 上币最快、小币和杠杆品种多              | 0% / 0.05%–0.10%（MX 持仓可额外折扣）                | 0.00% / 0.01%–0.02%（MX 持仓可额外折扣）             | 打新 / 猎奇小币用户            | 安全性争议较多                | 不支持 |
| HTX      | 老牌平台、流动性好、中文客服优秀        | 0.20% / 0.20%（HT 持仓可降 65%）                     | 0.02% / 0.05%–0.06%（HT / VIP 可降至负 Maker）       | 熟悉老平台的用户、合约玩家    | 历史监管事件、排名有所下滑    | 支持（P2P / C2C） |
| Kraken   | 安全性顶级、监管合规、储备证明透明      | 0.16%–0.25% / 0.26%–0.40%（Pro 卷量可降至 0% / 0.08%+） | 0.02% / 0.05%（卷量后可更低，但品种较少）            | 合规 / 安全性优先、机构 / 长期持有者 | 小币少、上币速度慢            | 不支持 |

### 推荐
- 全能型、追求最大流动性和最多币种 → **Binance**（大陆用户需注意访问方式和合规）。
- 合约交易、DeFi 与 CeFi 结合使用 → **OKX**（创新多、手续费低）。
- 打新、上小币、猎奇玩家 → **Gate.io** 或 **MEXC**（上币快、费用低/零，但需关注安全性）。
- 老牌平台、中文服务熟悉 → **HTX**。
- 极致安全性、合规优先、长期持有 → **Kraken**（费用较高，但安全与透明度顶级）。
- **通用建议**：交易量高=流动性更好，但结合自身需求（合约、小币、安全）选择；分散资产，避免单一平台风险；手续费可通过持币/VIP大幅降低。

## 量化交易框架对比

| 库/框架名称       | 主要语言                  | 支持交易所数量 | 是否支持实盘交易 | WebSocket 支持 | 社区与维护状态     | 核心特点与主要使用场景                              |
|-------------------|---------------------------|----------------|------------------|----------------|--------------------|-----------------------------------------------------|
| CCXT              | Python / JavaScript / TypeScript / PHP / Go / C# 等 | 100+           | 是               | 优秀（Pro 版更强） | 极其活跃、持续维护 | 目前加密货币程序化交易最通用、最全面的统一接口库，几乎所有量化项目的基础选择 |
| CCXT Pro          | 与 CCXT 相同              | 100+           | 是               | 极强（增强版）   | 高度活跃、专业维护 | CCXT 的付费增强版，提供更稳定、更低延迟的 WebSocket 和私有流支持 |
| Freqtrade         | Python                    | 通过 CCXT      | 是               | 依赖 CCXT        | 社区非常活跃       | 最受欢迎的开源加密货币交易机器人，支持策略编写、回测、实盘运行、参数优化、Telegram 通知等完整流程 |
| Hummingbot        | Python                    | 较多（部分自研 + CCXT） | 是            | 优秀             | 社区活跃、专业方向 | 专注于做市、跨交易所套利、AMM 流动性挖矿等专业场景的开源机器人 |
| cryptofeed        | Python                    | 20–40+         | 否（仅行情）     | 极致优秀         | 社区稳定且活跃     | 专注于高性能、低延迟的 WebSocket 行情采集，常用于对延迟敏感的策略 |
| python-binance / ccxt.binance 等官方库 | Python 等     | 仅 Binance     | 是               | 优秀             | 官方持续维护       | Binance 官方或社区维护的专用库，功能最完整、延迟最低 |
| Bybit / OKX / Binance 等官方 SDK | 多语言        | 仅自家交易所   | 是               | 优秀             | 官方高度维护       | 头部交易所官方提供的 SDK，延迟最低、接口最完整      |
| Vezgo             | 多语言 SDK（API 服务）    | 较多           | 否（只读）       | —                | 稳定运营           | 统一读取多交易所的资产余额、持仓、交易历史，主要用于资产追踪和展示 |

### 推荐
- 追求最多交易所支持 + 统一的交易接口 → **CCXT** 是目前最主流、最成熟的选择，几乎没有真正能全面替代它的开源项目。
- 需要极致实时性、稳定推送、私有流 → 升级到 **CCXT Pro**（付费，但专业团队常用）。
- 想要开箱即用的完整交易机器人系统（策略开发 → 回测 → 实盘 → 监控） → **Freqtrade** 是当前社区最活跃、最成熟的首选。
- 专注于做市、跨所套利、流动性提供 → **Hummingbot** 是最专业的方向性选择。
- 只需要超高质量、低延迟行情数据 → **cryptofeed** 是高性能行情采集的首选，常与 CCXT 组合使用。
- 只专注 1–2 家头部交易所（Binance、Bybit、OKX 等） → 优先使用**官方 SDK**，延迟更低、功能更完整、稳定性更高。

## 合约开发语言对比

| 语言名称     | 主要区块链 / 生态                | 语法风格 / 范式        | 学习曲线 | 安全性（内置机制）                  | 性能（执行效率） | 生态成熟度 / 工具支持 | 主流优势                              | 典型用例 / 缺点                              | 适合开发者类型                  |
|--------------|----------------------------------|------------------------|----------|-------------------------------------|------------------|-----------------------|---------------------------------------|--------------------------------------------|---------------------------------|
| Solidity    | Ethereum + 所有 EVM 链           | 类 C++ / JS，面向对象  | 中等     | 中等（需手动防重入、溢出等）        | 中等             | 极高                  | 生态最大、开发者最多、库最丰富        | DeFi、NFT、DAO；缺点：历史漏洞多、gas 优化复杂 | 大多数 Web3 开发者首选、EVM 玩家 |
| Vyper       | Ethereum + EVM 兼容链            | Python-like，简洁      | 中等 | 高（无重载、无继承、无无限循环）   | 中等             | 中等                  | 代码更安全、更易审计、更少漏洞        | 追求极致安全的协议（如稳定币、借贷）；缺点：功能受限、生态小 | 安全意识强、审计导向开发者      |
| Rust        | Solana（原生）、Polkadot、Near   | 系统级、所有权模型     | 高       | 高（内存安全、所有权借用防 bug）   | 极高             | 高                    | 极致性能 + 安全、适合高 TPS 链        | Solana DeFi / Memecoin / Game、基础设施；缺点：陡峭学习曲线 | 性能党、高并发开发者、Solana 重度用户 |
| Move        | Aptos、Sui（原生）               | 资源导向、类似 Rust    | 高       | 极高（资源所有权、线性类型、防重入）| 高               | 中高                  | 资产安全天生防双花 / 重入、最安全模型 | 高价值资产（如游戏物品、RWA）；缺点：生态还年轻、工具不如 EVM | 新兴 L1 开发者、注重资产安全者  |
| Cairo       | Starknet（ZK Rollup）            | Rust-like + ZK 友好    | 高       | 高（ZK 原生、安全模型强）          | 高（ZK 证明优化）| 中等                  | ZK 原生、隐私 + 扩展性强              | ZK DeFi、隐私应用、Layer2；缺点：ZK 知识门槛高 | ZK 技术爱好者、Starknet 开发者  |
| Anchor (Rust 框架) | Solana 专属                | Rust + 宏简化          | 高 | 高（继承 Rust 安全）               | 极高             | 高                    | 大幅降低 Solana 开发难度              | Solana 几乎所有 dApp；缺点：仍需懂 Rust 基础 | Solana 开发者主流选择           |

### 推荐
- 想快速进入 Web3 并接最多项目 → 优先学 **Solidity**（EVM 生态王者）。
- 专注高性能链如 Solana → 直接学 **Rust** + **Anchor**。
- 追求代码安全和审计友好 → 选 **Vyper** 或 **Move**。
- 对 ZK / 隐私感兴趣 → **Cairo** 是未来方向。
- 初学者从 Solidity 起步最稳，进阶后再扩展 **Rust** 或 **Move**。
