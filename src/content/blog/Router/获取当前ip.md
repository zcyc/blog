# 获取 IP

```bash
curl ifconfig.me
curl icanhazip.com
curl ipinfo.io/ip
curl ipecho.net/plain
curl ip.sb
curl whatismyip.akamai.com
curl ident.me
curl v4.ident.me
curl v6.ident.me (ipv6)
```

# 获取 IP 和位置

```bash
curl ip.cn
curl cip.cc
curl ipinfo.io
```
