# MPV
```
mpv --fs --fs-screen=1 --image-display-duration=5 --osd-level=0 --no-input-cursor --no-input-default-bindings --loop-playlist=inf --playlist=E:\mp4\playlist.m3u
```

# VLC
```
vlc --fullscreen --qt-fullscreen-screennumber=1 --image-duration=3000 --no-video-title-show --no-embedded-video --no-qt-fs-controller --loop E:\mp4\playlist.m3u
```
